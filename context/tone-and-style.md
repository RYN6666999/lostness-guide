# 語氣與風格指南

> 最後更新：2026-07-08
> 狀態：初版建立

---

## 風格核心

**直白、誠實、不雞湯、不高高在上。**

## 具體原則

### 1. 直白不繞圈
- 不要用華麗的開場鋪陳
- 不要用「讓我告訴你一個故事……」之類的套話
- 開門見山：「你不是沒有努力，你只是努力到不知道自己到底在逃什麼。」

### 2. 有故事，但不是炫耀
- 作者故事是為了讓讀者看見「我也經歷過」，不是為了證明作者很厲害
- 避免「後來我成功了」的敘事結構
- 多用「那時候的我……」、「後來我才理解……」、「現在回頭看……」
- 作者定位：一個曾經深度迷茫，正在用生命經驗與觀察，整理出校準系統的人。不是已經成功上岸的人。

### 3. 有洞察，但不堆術語
- 不要出現「量子糾纏」、「吸引力法則」、「高我」、「顯化」等詞
- 心理學或財務名詞要解釋，不要假設讀者懂
- 寧可用大白話，不要用專業名詞裝深

### 4. 對讀者誠實
- 不要承諾「看完這本書你就會找到方向」
- 要說實話：「方向不是找出來的，是校準後浮現的。而且校準是一輩子的事。」
- 誠實面對：有些問題沒有答案，有些改變需要時間

### 5. 可以尖銳，但不能羞辱
- 可以說：「你知道很多，但你做了什麼？」
- 不能說：「你就是太懶了。」
- 尖銳是為了讓讀者停下來想，不是為了貶低

### 6. 不要變成廣告
- ❌ 不要寫成房產課廣告
- ❌ 不要寫成釋放法傳教
- ❌ 不要寫成自傳
- ✅ 要寫成一套陪讀者重新校準人生的指南

## 語氣範例

### 不 OK
> 「在我生命的轉折點，我遇到了一位智慧的老師，他教會了我……從此我的人生就不一樣了。」

### OK
> 「我花了一百多萬上課，學了很多東西。後來我才發現，真正改變我的不是那些知識，而是我在過程中被迫面對自己的那些瞬間。」

### 不 OK
> 「只要你釋放你的限制性信念，宇宙就會給你最好的安排。」

### OK
> 「釋放法對我有用，但它不是萬能的。它有時候有用，有時候沒用。重要的是，它讓我開始注意到：我的情緒和念頭，不是我。」

## 寫作風格備忘

| 項目 | 原則 |
|------|------|
| 句子長度 | 短句為主，長句偶爾使用 |
| 段落 | 不超過 5–7 行，每個段落一個重點 |
| 語氣 | 像朋友認真跟你說話，不是講課 |
| 幽默 | 可以有，但不強求。不要硬搞笑 |
| 粗體 | 用於重點句，不要整段粗體 |
| 舉例 | 多用具體例子，少用抽象描述 |
| 反問 | 適度使用，讓讀者停下來想 |

## 章節開頭風格

每章開頭可以用一句「核心句」作為錨點，類似：

> 迷茫不是缺方向，而是道不明、術不熟、勢不察、器不足。

然後用一段話展開，不要鋪陳。

## 給 AI 寫作時的風格提示

當你需要 AI 協助寫作時，請附上：
1. 語氣要直白，不要雞湯
2. 不要用成功學語言
3. 作者是「走過迷茫的人」，不是「已經成功的人」
4. 重點是讓讀者感覺被理解，而不是被教育
