# 迷茫指南 — Lostness Guide

> 迷茫不是缺方向，而是道不明、術不熟、勢不察、器不足。

## 這是一本怎樣的書

《迷茫指南》不是一本教人找夢想、設目標、時間管理、正能量或成功學的書。

它寫給那些**很努力、很聰明、學很多，卻仍然卡住的人**。

這本書不是要告訴你「下一步做什麼」，而是要幫你重新校準——

- 把生命系統從「追逐答案」調回「感知方向」
- 讓你知道自己不是壞掉了，只是卡住了
- 讓迷茫從恐慌，變成可以共存的狀態

## Repo 目的

這個 repo 是《迷茫指南》的寫作基地。它存在的理由是：

1. **長期存放** — 所有素材、章節、決策、框架，都在這裡
2. **可維護** — 每一層都有清楚的結構，不會變成一堆草稿
3. **可協作** — 無論是自己寫、找 AI 協助、或未來找人合作，都有脈絡可循
4. **可追溯** — 每個決定都有紀錄，每個素材都有來源

## 目錄結構

```
lostness-guide/
├── book/          # 章節內容（目前為骨架階段）
├── context/       # 核心命題、目標讀者、風格、術語
├── materials/     # 素材庫：故事、觀察、筆記
├── framework/     # 核心框架：道術勢器、鬆開看清行動接住等
├── references/    # 外部參考資料庫（git submodule）
├── decisions/     # 架構決策記錄（ADR）
├── backlog/       # 待補問題、待辦事項、缺少素材
└── prompts/       # 給 AI 的寫作提示詞模板
```

## 核心框架速覽

| 框架 | 核心 |
|------|------|
| 道術勢器 | 底層心法、具體方法、世界流動、承接能力 |
| 鬆開→看清→行動→接住 | 釋放控制 → 認識自我與世界 → 小行動讓阻抗浮現 → 接住失敗長出篤定 |
| 聰明不是智慧 | 大腦能力 vs 生命系統的清明 |
| 財商是生命校準 | 金錢是面對現實最直接的修煉場 |
