# 迷茫指南 — Lostness Guide

> 用交通工具（覺察儀表板與引擎）讀懂地圖（道術勢器），把失準處校準，對齊後生出「以喜悅為引擎的篤定」，往前走——一生的修煉。

## 雙圖層（30 秒讀懂）

本書用兩張圖疊著看：**地圖**（道術勢器，回答「地形長怎樣」）＋**交通工具藍圖**（儀表板與引擎，回答「這台車靠什麼跑」）。

迷茫是儀表板的警示燈，不是故障本身——它是**訊號**，代表某個系統沒對齊。讀者的工作是用**覺察**讀懂這盞燈、對照地圖**診斷**哪裡沒對齊、把它**對齊**、對齊後生出**篤定**、帶著篤定**前進**，直到下一次不對齊出現，燈再亮，回到覺察。這是一個循環，一生的修煉，不是一次性解決。

真正的引擎是**喜悅**，不是恐懼、慣性或比較——那些也能推車，但只有喜悅是你自己的引擎。

完整版見 [`context/core-thesis.md`](context/core-thesis.md)。

## 這是一本怎樣的書

《迷茫指南》不是一本教人找夢想、設目標、時間管理、正能量或成功學的書。

它寫給那些**很努力、很聰明、學很多，卻仍然卡住的人**。

這本書不是要告訴你「下一步做什麼」，而是要幫你重新校準——

- 把生命系統從「追逐答案」調回「感知方向」
- 讓你知道自己不是壞掉了，只是卡住了
- 讓迷茫從恐慌，變成可以共存的狀態

## Repo 目的

這個 repo 是《迷茫指南》的寫作基地。它存在的理由是：

1. **長期存放** — 所有素材、章節、決策、框架，都在這裡
2. **可維護** — 每一層都有清楚的結構，不會變成一堆草稿
3. **可協作** — 無論是自己寫、找 AI 協助、或未來找人合作，都有脈絡可循
4. **可追溯** — 每個決定都有紀錄，每個素材都有來源

## 核心框架速覽

雙圖層是最上層的敘事骨架（見上方：地圖＋交通工具藍圖）；下面這些框架是兩張圖裡各自使用的具體工具，不是第三套說法：

| 框架 | 屬於哪張圖 | 核心 |
|------|-----------|------|
| 道術勢器 | 地圖 | 底層心法、具體方法、世界流動、承接能力 |
| 鬆開→看清→行動→接住 | 交通工具藍圖（校準循環的細節操作） | 釋放控制 → 認識自我與世界 → 小行動讓阻抗浮現 → 接住失敗長出篤定 |
| 聰明不是智慧 | 地圖（道的細節） | 大腦能力 vs 生命系統的清明 |
| 財商是生命校準 | 地圖（器的細節） | 金錢是面對現實最直接的修煉場 |

## 核心一句話

**用交通工具（覺察儀表板與引擎）讀懂地圖（道術勢器），把失準處校準，對齊後生出「以喜悅為引擎的篤定」，往前走——一生的修煉。**

（也可以說：迷茫是儀表板的警示燈，不是故障本身；也可以說：迷茫不是缺答案，而是生命系統失準——三句話說的是同一件事，只是解析度不同）
