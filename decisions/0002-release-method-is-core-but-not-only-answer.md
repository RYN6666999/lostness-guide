# 決策記錄 — ADR 0002

> 最後更新：2026-07-08
> 狀態：已確認

---

## 標題

釋放法是核心心法，但不是唯一解

## 決策

釋放法（Release Method / Sedona Method）是本書的一個重要工具，特別是在「鬆開」層面。但它不是全書的唯一核心。

## 理由

- 作者親身經驗中，釋放法確實有幫助
- 但一本書如果只推一個方法，會變成傳教
- 不同讀者適合不同方法——釋放法不是萬能的
- 本書的框架（道術勢器、鬆開看清行動接住）比單一方法更重要

## 影響

- 書中會介紹釋放法的基本概念與操作
- 但不會把它當成唯一的答案
- 會清楚說明釋放法的適用範圍與限制
- 會搭配其他方法一起使用

## 範疇

第 3 章（鬆開不是躺平）會重點使用釋放法，但其他章節不會以它為主軸。
