## 為什麼知己與知彼要放在一起

大部分人都只做其中一件事。

有的人一直在探索自己——上身心靈課、做測驗、寫日記、回顧童年創傷。
但他們對世界怎麼運作一無所知。

有的人一直在分析世界——看財經新聞、研究產業趨勢、分析公司財報。
但他們對自己的恐懼與限制視而不見。

**只知己不知彼** → 靈性逃避，無法落地。

**只知彼不知己** → 賺到錢但不快樂，或者有成就但內心空虛。

## 知己 — 真正了解自己

知己不是做人格測驗。

人格測驗（MBTI、人類圖、星盤）只能給你分類，不能給你真相。

真正的知己是：

### 1. 了解自己的身體
- 什麼時候最有能量？什麼時候最累？
- 壓力來的時候身體哪裡會緊？
- 哪些事情會讓你身體放鬆？

### 2. 了解自己的情緒
- 什麼事會讓你焦慮？
- 什麼事會讓你生氣？
- 什麼事會讓你躲起來？

### 3. 了解自己的潛意識模式
- 你總是重複什麼樣的關係模式？
- 你面對壓力時的反應是什麼？（戰、逃、僵、討好）
- 你最怕失去什麼？（安全感、認同、控制）

### 4. 了解自己的信念
- 你對金錢的信念是什麼？
- 你對自己的信念是什麼？
- 你對世界的信念是什麼？

## 知彼 — 了解世界怎麼運作

知彼不是查資料。

知彼是：

### 1. 理解金流
- 錢從哪裡來？往哪裡去？
- 什麼產業在成長？什麼在被淘汰？
- 誰在賺錢？他們做對了什麼？

### 2. 理解人流
- 人在往哪裡移動？（城市、產業、平台）
- 優秀的人在什麼環境？
- 你想成為的人，他們在哪裡？

### 3. 理解權力與規則
- 誰在訂規則？
- 什麼資源是稀缺的？
- 關係與信用值多少錢？

### 4. 理解時機
- 什麼時候該進？什麼時候該退？
- 什麼是長期趨勢？什麼是短期波動？
- 耐心 vs 行動的節奏

## 知己 × 知彼 的生長循環

```
知己 ────→ 知道自己要什麼
    ↘              ↙
     校準與行動
    ↗              ↘
知彼 ────→ 知道世界有什麼
```

當你同時知己與知彼，你的選擇就會從「猜」變成「校準」。
