# 統一校準模型（視覺圖）

> 最後更新：2026-07-08
> 狀態：初版建立，依 `backlog/questions.md` 的框架整合決定製作
> 對應素材：`framework/unified-model.svg`

---

## 用途

依作者在 `backlog/questions.md` 的決定：兩個框架（道術勢器 / 鬆開看清行動接住）要合併成一張圖，放在前言之後、第一章之前，讓讀者一眼看懂全書的兩層結構。

## 結構說明

- **外圈（結構診斷）**：道、術、勢、器——四個靜態維度，回答「你卡在哪一層？」
- **內圈（校準循環）**：鬆開→看清→行動→接住——可反覆使用的動態循環，回答「怎麼從卡住到能動？」

## 對應關係（詳見 `framework/loosen-see-do-catch.md`）

| 循環步驟 | 主要對應維度 | 說明 |
|---------|------------|------|
| 鬆開 | 道 | 放下控制、認同、安全感的束縛 |
| 看清 | 道 + 術 + 勢 | 同時看清自己（知己，偏道）與世界（知彼，偏勢），也開始盤點方法（術） |
| 行動 | 術 | 小行動，測試而非賭博 |
| 接住 | 器 | 承接失敗、不確定，需要身體、現金流、關係等「器」的支撐 |

圖中為求視覺簡潔，「看清」置於「勢」側，但實際涵蓋道、術、勢三層——這個簡化在 SVG 圖說腳註中已標明，正文引用此圖時應口頭補充這一點，避免讀者誤以為看清只等於知彼。

## 使用位置

- 前言之後、第一章之前，作為「本書框架速覽」單頁
- 每章骨架檔的「可用素材」可引用此圖，提醒該章屬於循環的哪一步、對應哪個維度
