# 觀察到的人群

> 最後更新：2026-07-08
> 狀態：初版建立（持續擴充中）

---

> 這些都不是虛構的類型。每一類都有作者親身或近距離觀察到的真實案例。

---

## 類型一：小白

| 項目 | 內容 |
|------|------|
| **表面狀態** | 對投資、買房、資產感到遙遠。不是不想懂，是覺得「這不是我的世界」。 |
| **真正卡點** | 距離感。不是笨，是沒有接觸的管道。從小家裡不談錢，學校不教財商，社會對這個領域的論述要麼太遙遠（有錢人的世界）、要麼太醜陋（炒房、割韭菜）。 |
| **故事方向** | 作者自己也曾經是小白的親身經歷。從「完全不懂」到「開始理解」的過程。 |
| **對應章節** | 06-財商是最現實的修煉場；05-知彼不是查資料 |

## 類型二：窮業務

| 項目 | 內容 |
|------|------|
| **表面狀態** | 很努力，很會講，但一直時間換錢，沒有槓桿。業績好時收入不錯，但沒有資產。 |
| **真正卡點** | 沒有建立系統。所有收入來自自己的時間與勞動力。沒有被動收入，沒有資產配置觀念。 |
| **故事方向** | 作者當業務時期的經歷。以及後來如何理解「業務可以不只是時間換錢」。 |
| **對應章節** | 06-財商是最現實的修煉場；08-行動是讓阻抗浮出來 |

## 類型三：亂投資的人

| 項目 | 內容 |
|------|------|
| **表面狀態** | 有行動力，但像賭徒。看到機會就衝，資金盤、幣圈、NFT、期貨什麼都碰。 |
| **真正卡點** | 太想翻身。內心有「現在還不夠」的急迫感，把每個機會都當救命繩。不是貪婪，是恐懼。 |
| **故事方向** | 作者 NFT 賠 24 萬、FTX 事件、同時做六件事的背景。以及後來如何理解「慢才是快」。 |
| **對應章節** | 06-財商是最現實的修煉場；02-聰明人的陷阱；09-接住失敗才會長出篤定 |

## 類型四：高薪受雇者

| 項目 | 內容 |
|------|------|
| **表面狀態** | 月收入高（8–15 萬），看似人生勝利組。但存不下錢，或不敢做重大財務決策。 |
| **真正卡點** | 有收入，不代表有財商。舒適圈也是一種陷阱——穩定的高薪讓人麻痹，失去了看懂現實規則的動力。同時，高生活水準會讓開銷跟著拉高。 |
| **故事方向** | 觀察到的高薪朋友案例。以及作者對「高薪 ≠ 自由」的反思。 |
| **對應章節** | 06-財商是最現實的修煉場；07-環境教練工具如何重塑人 |

## 類型五：接案高薪者

| 項目 | 內容 |
|------|------|
| **表面狀態** | 有能力賺錢（月收入 10–30 萬），但收入不穩，缺少資產與槓桿觀。 |
| **真正卡點** | 很會賺，但不會留。沒有資產配置觀念，也沒有建立被動收入系統。對風險的認識不足。 |
| **故事方向** | 自由接案者如何建立財務安全感？如何從純勞動力收入轉向資產收入？ |
| **對應章節** | 06-財商是最現實的修煉場；05-知彼不是查資料 |

## 類型六：受害者

| 項目 | 內容 |
|------|------|
| **表面狀態** | 曾被直銷、資金盤、投資坑過，所以高度防衛。 |
| **真正卡點** | 創傷沒有被處理。不是笨才被騙，是在某個脆弱的時刻遇到對的包裝。受傷之後，從「太相信人」變成「誰都不信」，兩種極端都不是校準。 |
| **故事方向** | 作者被坑過的經歷。以及如何從創傷中恢復對世界的信任（但不再是天真）。 |
| **對應章節** | 09-接住失敗才會長出篤定；02-聰明人的陷阱 |

## 類型七：信徒

| 項目 | 內容 |
|------|------|
| **表面狀態** | 深陷某套信念系統。可能是成功學、靈性逃避、投資神話、努力迷信。 |
| **真正卡點** | 把一個工具當成了全部答案。不是那個工具不好（直銷有直銷的技術、釋放法有釋放法的價值），而是把局部當整體。缺乏對自己生命系統的全面審視。 |
| **故事方向** | 作者自己曾經是信徒（直銷、身心靈、各種系統的狂熱追隨者）。以及後來如何從信徒變成使用者——用工具，但不被工具綁住。 |
| **對應章節** | 03-鬆開不是躺平；02-聰明人的陷阱 |

---

## 給自己的提醒

- 這些類型是原型，不是標籤。
- 一個人可以同時屬於多種類型。
- 寫作時避免用「你是 OO 型的人」這種分類語言。
- 更好的方式是：「你可能遇過這樣的人……或者你曾經就是這樣。」
