# 待補問題

> 最後更新：2026-07-08
> 狀態：初版建立
> 用途：記錄寫作過程中需要作者補上的資訊、決定、或釐清的問題

---

## 關於書的本質

- [ ] 這本書的目標長度？（10 萬字？15 萬字？20 萬字？）
- [ ] 出書方式：電子書？實體書？連載？自出版？
- [ ] 是否需要搭配網站或線上資源？
- [ ] 是否需要練習本或工作簿？
- [ ] 是否需要建立讀者社群？

## 關於作者定位

- [ ] 作者在書中的「人設」要維持什麼層次？
  - 是「曾經迷茫的人」還是「正在走出迷茫的人」？
  - 書中要不要承認作者現在也還在過程中？
- [ ] 作者的真實身份要揭露多少？（姓名、職業、現況）

## 關於內容

- [ ] 書中的案例：哪些要用作者真實姓名？哪些要匿名？
- [ ] 是否需要引用研究、論文、或心理學理論來增加權威性？
- [ ] 台灣本土的財商數據要去哪裡找？
- [ ] 釋放法以外的「鬆開」方法有哪些值得介紹？

## 關於框架

- [ ] 「道術勢器」和「鬆開看清行動接住」兩個框架的關係要不要在書中明確定義？
- [ ] 兩個框架是否要合併成一個統一的圖示或模型？
- [ ] 要不要有「本書框架速覽」之類的單頁指南？

## 關於風格

- [ ] 每章結尾要不要有「本章練習」？
- [ ] 要不要有「給讀者的話」或「作者後記」？
- [ ] 要不要放參考書單？

## 關於時程

- [ ] 第一階段的承載系統建立後，第二階段的優先級是什麼？
- [ ] 每個章節的寫作順序？從最容易的開始還是最核心的開始？
- [ ] 目標完成初稿的時間點？

## 關於實驗

- [ ] 要不要先寫幾章發在網路上測試讀者反應？
- [ ] 要不要先做一個 mini version（小冊子）測試市場？
