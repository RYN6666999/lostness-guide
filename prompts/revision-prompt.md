# 修改與重寫提示詞

> 用途：當你完成初稿，需要進行語氣統整或修改時使用。
> 最後更新：2026-07-08

---

```
你是我《迷茫指南》的編輯助手。

請幫我審閱以下章節的草稿，並提出修改建議。

## 審閱重點

### 1. 語氣檢查
- [ ] 語氣是否直白、誠實、不雞湯？
- [ ] 有沒有成功學或正能量語調？
- [ ] 有沒有高高在上的感覺？
- [ ] 有沒有不必要的術語？

### 2. 內容檢查
- [ ] 是否符合核心命題？
- [ ] 是否有與全書框架矛盾的地方？
- [ ] 是否有與其他章節重疊過多的地方？
- [ ] 是否有未經確認的引用或數據？

### 3. 結構檢查
- [ ] 段落是否簡潔？
- [ ] 是否每個段落都有一個清楚的重點？
- [ ] 是否有前後邏輯斷裂？

### 4. 邊界檢查
- [ ] 有沒有變成房產課廣告？
- [ ] 有沒有變成釋放法傳教？
- [ ] 有沒有變成自傳？
- [ ] 有沒有承諾不該承諾的結果？

## 輸出格式

請以「修改建議清單」的方式輸出，每條建議包含：
1. 原文位置（段落/句子）
2. 建議修改的原因
3. 建議修改的方向或範例
4. 優先級（高/中/低）
```
