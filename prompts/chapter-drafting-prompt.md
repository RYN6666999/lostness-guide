# 章節試寫提示詞

> 用途：當你準備好開始寫某一個章節的正文時使用。
> 最後更新：2026-07-08

---

```
你是我《迷茫指南》的寫作助手。

請為第 [X] 章「[章節標題]」撰寫正文。

## 核心要求

請先讀取以下檔案來了解全書脈絡：
- context/core-thesis.md（核心命題）
- context/target-reader.md（目標讀者）
- context/tone-and-style.md（語氣風格）
- framework/（相關的框架文件）
- materials/（相關的素材文件）
- book/XX-章節檔.md（該章的骨架）

## 寫作規範

1. 語氣要直白，不要雞湯
2. 不要用成功學語言
3. 不要用「你一定可以」、「改變你的一生」這類承諾
4. 作者是「走過迷茫的人」，不是「已經成功的人」
5. 重點是讓讀者感覺被理解，而不是被教育
6. 短句為主，段落不超過 5–7 行
7. 不要堆術語，不要用「量子○○」之類的偽科學詞彙
8. 不要寫成房產課廣告、釋放法傳教、自傳

## 本章已確認的內容

（填入該章骨架檔中的可用素材、結構、關鍵金句）

## 請輸出

1. 本章正文（3000–5000 字）
2. 寫作過程中遇到的問題或需要我確認的地方
3. 建議的下一章
```
