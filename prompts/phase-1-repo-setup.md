# Phase 1：Repo 建立提示詞

> 用途：當你需要一個新的 AI 實例來建立或還原這個 repo 時使用。
> 最後更新：2026-07-08

---

```
你是我的本地寫作工程助手。

我要建立一個 repo，用來長期存放、整理、更新、撰寫一本書：《迷茫指南》。

請你先不要急著寫正文。

第一階段任務是：建立一個乾淨、可長期維護的 repo 架構，讓這本書未來可以逐步補完。

這本書目前的核心定位：

《迷茫指南》不是一本教人找夢想、設目標、時間管理、正能量或成功學的書。
它寫給那些很努力、很聰明、學很多，卻仍然卡住的人。

核心句：迷茫不是缺答案，而是生命系統失準。
核心框架：道術勢器、鬆開看清行動接住、聰明不是智慧、財商是生命校準。

請依照目錄結構建立完整的承載系統：

lostness-guide/
├── README.md
├── book/           # 章節骨架（只放大綱，不寫正文）
├── context/        # 核心命題、目標讀者、風格、術語
├── materials/      # 素材庫
├── framework/      # 核心框架
├── decisions/      # 架構決策
├── backlog/        # 待辦與問題
└── prompts/        # 提示詞模板

重要限制：
1. 不要寫完整章節
2. 不要發明新理論
3. 作者定位是「走過迷茫的人」，不是「成功上岸的人」
4. 所有內容使用 markdown
5. 每個檔案要保持清楚、可維護、可逐步擴充
```
